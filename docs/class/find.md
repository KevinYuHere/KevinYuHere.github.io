# 对班级管理中的突出问题的调查

## 前言

我们对班级管理中的突出问题做了调查，以下是有价值的回答。

对话双方及提及的同学都是化名，且部分对话经过省略、模糊或替换。

> A
> 对呀，这就是一个很明显的问题，C同学也没有水，也没有很严格，就只是正常的中庸，然后公平公正吧，然后大家对TA没有什么意见吧，然后D同学就很明显就是你有一个小错误（跟正事完全无关的那种）都要算错，这个很明显，大家肯定不能接受吧。
